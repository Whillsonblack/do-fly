doefly

Easy to use css framework for all without restrictions

The syntax of doefly is very simple. Each class meets its intended purpose. This product was developed exclusively for personal purposes. It is improving and improving.

The ideology of doefly is customization. The value is not in the finished buttons or other elements. You can make any of them perfect for yourself! This causes the HTML code to load, which is not suitable for everyone. We understand this and recommend using doefly for small projects, such as the landing page.

The name of the class is its value

For example:

  • .colorWhite -> {color: #ffffff;}
  • .marginNone -> {margin: 0;}
  • .paddingNone -> {padding: 0;}
  • .textDecNon -> {text-decoration: none;}
  • .bradius3 -> {border-radius: 3px;}
  • .bradiusTopLeft3 -> {border-radius: 3px 0 0 0;}
  • .bradiusLeft3 -> {border-radius: 3px 0 0 3px;}
  • .wrap1200 -> {width: 960px; margin: 0 auto;}
  • .fixed -> {position: fixed;}
  • .width100Per -> {width: 100%;}
  • .width100Px -> {width: 100px;}
  • .height100Per -> {height: 100%;}
  • .height100Px -> {height: 100px;}
  • .hoverColorBlack -> :hover{color: #000000;}
  • .hoverBgBlack -> :hover{background: #000000;}
  • .fontSize13 -> {font-size: 13px;}
  • .margin5 -> {margin: 5px;}
  • .marginTop5 -> {margin-top: 5px;}
  • .padding5 -> {padding: 5px;}
  • .paddingTop5 -> {padding-top: 5px;}
  • .dInline -> {display: inline;}
  • .dNone -> {display: none;}
  • .listStyleNone -> {list-style: none;}
  • .ulInlineBlock -> > li {display: inline-block;}
  • .clearFix -> {clear: both;}
  • .alignCenter -> {text-align: center;}
  • .floatLeft -> {float: left;}
  • .pTop50Per -> {top: 50%;}
  • .defaultTransition -> {transition: all .3s;}
  • .borderTopRedSolid -> {border-top: 2px solid #db2828;}
  • .bold -> {font-weight: bold;}
  • .bgCover -> {background-size: cover;}
  • .zIndex100 -> {z-index: 100;}
  • .blackTextShadow2 -> {text-shadow: 2px 2px #000000;}
